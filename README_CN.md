## 基本介绍：

这是Log Service SDK for Python的开源版本。Log Service SDK for Python是阿里云日志服务
（Log Service）API的Python编程接口，提供了对于Log Service Rest API所有接口的封装
和支持，帮助Python开发人员更快编程使用阿里云Log Service服务。

### 具体功能：

1. 封装Rest API。
2. 实现API请求的数字签名
3. 实现API的Protocol Buffer格式发送日志
4. 支持API定义的数据压缩方式
5. 实现API查询数据和批量消费数据
6. 使用异常统一处理错误
7. 提供消费组高级API
